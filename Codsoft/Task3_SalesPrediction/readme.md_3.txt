# 📈 Sales Prediction Using Python

This project aims to predict product sales using machine learning techniques in Python. It explores how factors like advertising expenditure, audience targeting, and platform selection affect sales.

## 🎯 Objective

Forecast future sales to help businesses make data-driven decisions, optimize advertising strategies, and improve revenue.

## 📄 Features Used

- `TV`, `Radio`, `Newspaper` advertising spend
- `Product Type`
- `Target Audience`
- `Sales` (target variable)

## 🧼 Preprocessing

- Handled missing values
- Encoded categorical variables (if any)
- Performed scaling and feature selection

## 🤖 Model Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

**Best model:** `Random Forest Regressor`  
**R² Score:** 0.89  
**RMSE:** 1.27

## ▶️ How to Run

1. Install dependencies:  
   `pip install -r requirements.txt`
2. Run the notebook:  
   `jupyter notebook Task3_SalesPrediction.ipynb`
